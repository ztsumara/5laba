Краткая документация

Get запросы

  • /home - корневой путь, res = Hello, its a home page
  • /starts, res = количество запросов на данный путь
  • /comments, res = вывод всех документов в бд
  • /comments/:id?, res = вывод документа с конкретным именем

Post запросы

  • /comments, res = пост запрос в ормате json, отправка данных в бд
